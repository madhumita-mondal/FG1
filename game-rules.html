🔢 Number Path

Start from the blue current number.
Connect to the yellow target number.
Follow targets in order.
Do not cross or touch previous paths.
Use Undo or Hint if needed.
Finish before the timer ends.
🎉 Complete the entire number path.

🧩 Sliding IQ Puzzle

🟢 Easy Mode
Arrange numbers 1–11 in order.
Only tiles next to the empty space can move.
Use fewer moves for a higher IQ score.
🏆 Arrange 1–11 correctly.
🟡 Medium Mode
Arrange numbers 1–15 in order.
Only adjacent tiles can move.
Faster completion earns a better IQ score.
🏆 Arrange 1–15 correctly.
🟣 Encouragement Mode
Reveal the hidden encouragement message.
Use Row & Column clues on each tile.
Only adjacent tiles can move.
The original message is shown above the board.
🌟 Reveal the complete message.

Word Match Arena

Add or select a player before starting the game.
Choose a mode: Easy, Medium, Hard, or Tough.
Press Start Game to get a root word.
Make words that start with one of the root word letters.
Do not enter the root word itself. It gives 0 points.
Do not repeat the same word in one round.
Use letters only.
Scoring Rules
Exact hidden system word = 15 points.
Correct listed word = 2 points.
Weak word = 1 point.
Root word = 0 points.
Two-letter words or shorter = 0 points.
Medium, Hard, and Tough modes give bonus points for longer correct words.
Timer
Easy Mode: 90 seconds.
Medium Mode: 75 seconds.
Hard Mode: 65 seconds.
Tough Mode: 55 seconds.
Buttons
Submit checks your word and adds points.
Reveal All shows the hidden system words.
Restart starts a new round.
Clear All Scores resets the scoreboard.
Score as many points as possible before the timer ends.

Math Monopoly

4 players can play the game.
Each player starts with $700.
Players take turns rolling the dice and moving around the board.
Passing START gives $100.
Property Rules
Landing on an unowned property gives a math challenge.
Solve correctly to buy that property.
If you land on your own property, nothing happens.
Landing on another player's property means paying rent.
Correct answer = pay half rent.
Wrong answer = pay full rent.
Special Tiles
Chance tiles give rewards or penalties.
Tax tiles require math challenges.
Correct answer on tax tile = reduced tax.
Wrong answer on tax tile = full tax.
GO TO JAIL sends the player to jail.
Solve a challenge correctly to escape jail.
Math Categories
Questions include Fractions, Decimals, Multiplication, Division, and Geometry.
Fractions should be typed like 3/4.
Decimals should be typed like 2.5.
Build the highest net worth to win the game.

🕵️ Crook Chase

4–10 players can play.
Enter all player names and start the round.
System shuffles roles automatically.
Role Distribution
4–6 players: 1 Cop, 1 Thief, 1 Robber, rest Civilians.
7–8 players: 2 Cops, 2 Thieves, 1 Robber, rest Civilians.
9–10 players: 2 Cops, 2 Thieves, 2 Robbers, rest Civilians.
How to Play
At round start, only Cop and Civilian coins are revealed.
Thief and Robber remain hidden.
Each Cop guesses one Thief and one Robber.
After guesses, hidden roles are revealed.
Scoring
Each Civilian gets +600 points.
Cop gets +500 if both guesses are correct.
Otherwise Cop gets +300.
Caught Thief gets -500, escaped Thief gets -400.
Caught Robber gets -600, escaped Robber gets -500.
Score the highest points after multiple rounds to win.

🎯 Hangman Blast

Game Modes
1. Word Hangman
2. Number Hangman
Word Hangman Rules
1–6 players can play.
Enter player names and start the game.
A hidden word appears as blank boxes.
Players guess one letter at a time.
Correct letters reveal their positions.
Wrong letters are added to the wrong list.
Every 3 wrong guesses adds 1 hangman body part.
Reveal the full word before hangman completes.
Hint button gives clue for the word.
Number Hangman Rules
Guess the secret number digit by digit.
Correct digit earns +10 points.
Correct digit in correct position earns +15 points.
Wrong digit loses -5 points.
Duplicate correct digit-position gives no extra points.
Winning
Complete the word or number with the highest score.
Player with highest total score wins.
Use logic, memory and smart guessing to win Hangman Blast!

🕒 Dashtime

1–10 players can play.
Enter player names and choose total rounds.
Available rounds: 5, 8, 10, or 15.
How to Play
Press Start Game to begin.
A clock with random hour and minute hands appears.
Observe the clock carefully.
Choose the correct hour and minute.
Minutes appear in 5-minute intervals only.
Press Submit Answer to lock your answer.
Buttons
Shuffle Clock generates a new clock.
Reveal Time shows the correct time.
Next Turn moves to the next player.
Reset Game clears scores and restarts.
Scoring
Correct hour = +10 points.
Correct minute = +10 points.
Exact time (hour + minute both correct) = extra +5 bonus.
Score the highest points by reading clocks accurately and quickly.
