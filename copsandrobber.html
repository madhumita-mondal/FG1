Crook Chase

Set up players, read the rules, then start the animated role game.

🎮 Game Setup

📜 Rules

👮 Cops guess one Thief and one Robber.
🧑 Civilians get +600 points every round.
👮 Cop correct both = +500, otherwise +300.
🕵️ Thief caught = -500, escaped = -400.
💰 Robber caught = -600, escaped = -500.

🔢 Role Distribution

4–6: 1 Cop, 1 Thief, 1 Robber, rest Civilians
7–8: 2 Cops, 2 Thieves, 1 Robber, rest Civilians
9–10: 2 Cops, 2 Thieves, 2 Robbers, rest Civilians
