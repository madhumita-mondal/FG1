🎯 Hangman Blast

Game Modes
1. Word Hangman
2. Number Hangman
Word Hangman Rules
1–6 players can play.
Enter player names and start the game.
A hidden word appears as blank boxes.
Players guess one letter at a time.
Correct letters reveal their positions.
Wrong letters are added to the wrong list.
Every 3 wrong guesses adds 1 hangman body part.
Reveal the full word before hangman completes.
Hint button gives clue for the word.
Number Hangman Rules
Guess the secret number digit by digit.
Correct digit earns +10 points.
Correct digit in correct position earns +15 points.
Wrong digit loses -5 points.
Duplicate correct digit-position gives no extra points.
Winning
Complete the word or number with the highest score.
Player with highest total score wins.
Use logic, memory and smart guessing to win Hangman Blast!
