🔤 Word Match Arena

Add or select a player before starting the game.
Choose a mode: Easy, Medium, Hard, or Tough.
Press Start Game to get a root word.
Make words that start with one of the root word letters.
Do not enter the root word itself. It gives 0 points.
Do not repeat the same word in one round.
Use letters only.
Scoring Rules
Exact hidden system word = 15 points.
Correct listed word = 2 points.
Weak word = 1 point.
Root word = 0 points.
Two-letter words or shorter = 0 points.
Medium, Hard, and Tough modes give bonus points for longer correct words.
Timer
Easy Mode: 90 seconds.
Medium Mode: 75 seconds.
Hard Mode: 65 seconds.
Tough Mode: 55 seconds.
Buttons
Submit checks your word and adds points.
Reveal All shows the hidden system words.
Restart starts a new round.
Clear All Scores resets the scoreboard.
Score as many points as possible before the timer ends.
