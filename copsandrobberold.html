🚓 Cop Guessing Zone

Only cops should use this section. Each cop selects one thief guess and one robber guess.

📜 Rules

1. Enter Names

Choose 4 to 10 players and enter all names.

2. Shuffle Roles

System assigns roles automatically each round.

3. Public Coins

Cop and Civilian names appear first on animated coins.

4. Cop Guess

Each cop selects one thief and one robber.

5. Civilian Points

Every civilian gets +600.

6. Cop Points

Correct thief + robber = +500, otherwise +300.

7. Thief Points

If caught by any cop: -500, otherwise -400.

8. Robber Points

If caught by any cop: -600, otherwise -500.

🔢 Role Distribution

  • 4–6 players: 1 Cop, 1 Thief, 1 Robber, rest Civilians
  • 7–8 players: 2 Cops, 2 Thieves, 1 Robber, rest Civilians
  • 9–10 players: 2 Cops, 2 Thieves, 2 Robbers, rest Civilians
